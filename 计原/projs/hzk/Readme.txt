"hzdisp" folder: including the executable file "hzDisp.exe"
"hzdisp_qtproj" folder: the qt project file

Start:
Enter the "hzdisp" folder;
Double click the executable file "hzDisp.exe" to run the program;
The initial interface will display a chinese charachter "永";
Input one chinese character in the input box;
Click the "hello" button and the "永" will be replaced by what you input;
Five additional function buttons: 
	larger, smaller, italic, rotate, color;
